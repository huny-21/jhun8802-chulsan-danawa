사이트 사용 방법

사용 방법 출산휴가 육아휴직 면책/추정치

가장 빠른 확인 순서는 우리동네 혜택 → 육아휴직 급여 계산기 → 육아휴직 달력 입니다. 이 순서로 진행하면 정부지원금 연동, 급여 계산, 일정 확인까지 한 번에 이어서 점검할 수 있습니다.

특히 동네 혜택 조회를 먼저 완료해야 계산기에서 정부지원금(출산 혜택 포함) 연동이 활성화됩니다. 또한 계산기에 입력한 출산휴가/육아휴직 정보는 달력에 기본값으로 자동 반영되어, 일정 확인 단계에서 재입력 부담을 줄일 수 있습니다.

1) 먼저, 기능별로 꼭 필요한 입력값

기능필수 입력입력 기준결과에서 볼 것
우리동네 혜택시/도, 시/군/구, 출산(예정)일, 아이 순위출산일은 최근 3년~향후 2년 범위, 아이 순위는 첫째/둘째/셋째 이상 기준으로 선택정부 공통/지자체 특화 분리, 총 예상 지원금
육아휴직 계산기월 통상임금, 출산전후휴가 시작일, 육아휴직 구간통상임금은 세전 기준, 구간은 실제 사용 예정일 기준(분할 가능)월별 지급액, 부분월(일할 계산) 표시, 추정/확정 모드
육아휴직 달력출산휴가/육아휴직 시작일·일수계산기와 동일한 구간을 달력에서 조정 가능월별 일정 충돌, 복직 예정일, 공휴일 포함 일정

2) 계산 결과를 읽을 때 기준이 되는 값

항목기준비고
출산전후휴가 기간기본 90일, 다태아 120일, 미숙아 100일입력값에 따라 자동 반영
출산전후휴가 급여 상한고용보험 구간 월 210만원우선지원대상기업 여부에 따라 회사/정부 구간 추정
육아휴직 일반형1~3개월 100%(상한 250), 4~6개월 100%(상한 200), 7개월 이후 80%(상한 160)하한 월 70만원
6+6 부모육아휴직제1~6개월 상한 250/250/300/350/400/450부모 각각 3개월 이상, 자녀 18개월 이내 등 요건 충족 시 적용
부분월 계산월 중 시작/종료 시 사용일 비율로 일할 계산그래프 `◐` 마커로 표시

3) 실제 입력 순서(권장)

  1. 우리동네 혜택에서 지역/출산일/아이 순위를 입력해 혜택 조회 완료
  2. 육아휴직 급여 계산기에서 정부지원금 연동 옵션 활성화 확인 후 월별 급여 계산
  3. 출산휴가 시작일과 육아휴직 구간을 입력해 월별 표·차트 확인
  4. 육아휴직 달력에서 자동 반영된 일정으로 겹침/복직일 최종 점검

4) 연결 사용 안내(중요)

서비스는 순차 연동을 권장하지만, 각 서비스는 단독으로도 사용할 수 있습니다. 예를 들어 계산기만 사용해 급여 추정치를 확인하거나, 달력만 사용해 일정 계획을 먼저 세울 수 있습니다. 다만 정부지원금까지 포함한 통합 결과를 보려면 동네 혜택 조회를 먼저 진행하는 것을 권장합니다.

공식 확인: 고용24, 고용보험, 정부24, 복지로
