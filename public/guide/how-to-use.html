사용 방법 출산휴가 육아휴직 면책/추정치

이 문서는 기능 설명보다 실사용 순서에 집중합니다. 처음 사용하는 경우에도 아래 순서대로 입력하면 “무엇을 입력해야 하는지”와 “결과를 어떻게 해석해야 하는지”를 빠르게 파악할 수 있습니다.

1) 먼저, 기능별로 꼭 필요한 입력값

기능필수 입력입력 기준결과에서 볼 것
우리동네 혜택시/도, 시/군/구, 출산(예정)일, 아이 순위출산일은 최근 3년~향후 2년 범위, 아이 순위는 첫째/둘째/셋째 이상 기준으로 선택정부 공통/지자체 특화 분리, 총 예상 지원금
육아휴직 계산기월 통상임금, 출산전후휴가 시작일, 육아휴직 구간통상임금은 세전 기준, 구간은 실제 사용 예정일 기준(분할 가능)월별 지급액, 부분월(일할 계산) 표시, 추정/확정 모드
육아휴직 달력출산휴가/육아휴직 시작일·일수계산기와 동일한 구간을 달력에서 조정 가능월별 일정 충돌, 복직 예정일, 공휴일 포함 일정

2) 계산 결과를 읽을 때 기준이 되는 값

항목기준비고
출산전후휴가 기간기본 90일, 다태아 120일, 미숙아 100일입력값에 따라 자동 반영
출산전후휴가 급여 상한고용보험 구간 월 210만원우선지원대상기업 여부에 따라 회사/정부 구간 추정
육아휴직 일반형1~3개월 100%(상한 250), 4~6개월 100%(상한 200), 7개월 이후 80%(상한 160)하한 월 70만원
6+6 부모육아휴직제1~6개월 상한 250/250/300/350/400/450조건 충족 시 적용
부분월 계산월 중 시작/종료 시 사용일 비율로 일할 계산그래프 `◐` 마커로 표시

3) 실제 입력 순서(권장)

  1. 출산전후휴가 시작일을 먼저 입력하고 종료일 자동 계산값 확인
  2. 본인 육아휴직 구간을 먼저 입력한 뒤 필요 시 배우자 구간 추가
  3. 통상임금/기업 유형을 확인해 추정치 오차를 줄이기
  4. 달력에서 일정 겹침과 복직 시점을 확인한 뒤 월별 표·차트 해석

4) 결과 해석 시 꼭 확인할 점

화면의 금액은 입력값 기준 예상치입니다. 실제 지급은 회사 급여 규정, 신청 시점 고시, 관할 기관 심사 결과에 따라 달라질 수 있습니다. 신청 직전에는 최신 공고를 확인하고 같은 조건으로 한 번 더 재계산하는 것을 권장합니다.

공식 확인: 고용24, 고용보험, 정부24, 복지로
